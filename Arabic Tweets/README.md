## Arabic Tweets

A corpus of Arabic tweets categorized to positive and negative sentiments based on some emoji characters appearance

Category        	  | Tweets No.  |
-----        	  | --------             |
Positive Tweets |   2,104,671   |
Negative Tweets |   2,313,457   |

### Dataset Description
every data record contains:
1. Tweet ID
2. Username
3. User ID
4. Tweet Date
5. Tweet Text
